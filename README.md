## üõ†Ô∏è Installation & Usage

### Prerequisites
- Modern web browser (Chrome, Firefox, Edge, Safari)
- Optionally, a local web server for advanced features

### Steps
1. **Clone the repository:**
	```sh
	git clone https://github.com/mohammedshaibaaz/ScaleNow.git
	```
2. **Open the project folder:**
	Navigate to the project directory and open `index.html` in your browser or preferred development environment.
3. **Customize assets and content:**
	- Add your own images to `assets/images/`
	- Update text and sections in `index.html`
4. **Edit styles and scripts:**
	- Modify `css/styles.css` for branding and layout
	- Update `js/main.js` for custom interactivity
